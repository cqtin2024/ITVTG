Cấu Hình & Đồng Bộ Hóa (GitHub API)

1. GitHub Personal Access Token (PAT)

CẢNH BÁO BẢO MẬT: Token này sẽ được lưu trữ trong trình duyệt. Chỉ sử dụng Token có phạm vi (Scope) nhỏ nhất. CẦN PHẢI CÓ QUYỀN **repo** (hoặc contents:write).

2. Cập Nhật Tỉ Số & Trạng Thái Toàn Bộ (Commit) lên state301.json

Sử dụng Token đã lưu để commit **TOÀN BỘ** trạng thái giải đấu (Tỉ số Vòng Bảng, Vòng Loại) lên file **state301.json**.

3. Commit Dữ Liệu Vòng Loại (Ghi đè state301.json)

Ghi đè toàn bộ dữ liệu (tỉ số & kết quả) của Vòng Loại trực tiếp (`Knockout` Tab) và trạng thái lên file **state301.json**.

4. Tùy Chọn Giao Diện & Đồng Bộ Lại

Bạn có thể chuyển đổi giữa giao diện Sáng/Tối hoặc tải lại trạng thái online mới nhất.
